Description:

User's browser family:

I implemented two different methods to find the information about User's browser family.

The first method:
Parsing the characters obtained from the User Agent via navigator.userAgent. Each browser will have information in the User Agent that contains browser features such as browser's name, browser's version, etc. I rely on these features to distinguish between browsers and give information about the User's browser family. List of Browsers that First Method Support: Chrome (Desktop & Mobile version), Firefox (Desktop version), Safari (Desktop & Mobile version), Opera (Desktop version), Internet Explorer and Edge.

The second method:
Distinguishes based on the unique features of each browser with Javascript Objects (Windows Object and HTML DOM Document Object). List of Browsers that Second Method Support (Desktop version only): Chrome, Firefox, Safari, Opera, Internet Explorer and Edge.

In addition, I also fingerprint the following information:
Browser's Version (based on User Agent)
Number of plugins installed (based on navigator Object)
List of Plugins (based on navigator Object)
Operating System (based on User Agent) - Support: Microsoft Windows, MacOS, Linux, Android, iOS
Cookies enable (based on navigator Object)
Screen Resolution (based on screen object)
Color Resolution (based on screen object)
Browser's Language (based on Navigator Object)

Identifier:
The identifier is based on extracting special characters from the acquired information of Browser's Name, Browser's Version, Operating System, Number of plugins, Cookies enabled, Screen Resolution and Browser's Language.
